HTML Block vs Inline Elements

1. HTML Tag Types: Paired vs Self-Closing

HTML elements are either paired tags or self-closing tags:

Paired Tags (Start and End Tags)

Most elements have a start tag and an end tag:

<p>This is a paragraph</p>
<div>Container</div>
<h1>Heading</h1>
<ul><li>Item</li></ul>

Explanation: Paired tags enclose content and define its meaning.

Self-Closing Tags

Some elements do not have content and are written as self-closing:

<br />  Line break
<hr />  Horizontal line
<img src="image.jpg" alt="image" />  Image
<input type="text" />  Input field
<meta charset="UTF-8" />  Metadata
<link rel="stylesheet" href="style.css" />  Stylesheet link

Explanation: Self-closing tags do not wrap content and always end with />. Modern HTML allows just <br> without slash, but / is explicit.

2. Headings: <h1> - <h6>

Headings define document hierarchy. They are block elements and start on a new line:

Code:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Live Output:

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
Explanation: Headings are block elements, always starting on a new line, and help structure your page for readability and SEO.

3. Common Block Elements

Block elements occupy full width and start on a new line. They can contain inline and other block elements:

Code:

<div>Container</div>
<section>Section</section>
<article>Article</article>
<header>Header</header>
<footer>Footer</footer>
<nav>Navigation</nav>
<pre>Preformatted text</pre>
<blockquote>Quote</blockquote>
<hr /> Horizontal line
<table><tr><td>Cell</td></tr></table>
<ul><li>Item</li></ul>
<ol><li>Item</li></ol>
<form>Form elements</form>

4. Common Inline Elements

Inline elements stay inside the text flow and occupy only the width of their content:

Code:

<span>Text</span>
<a href="#">Link</a>
<strong>Strong</strong>
<b>Bold</b>
<em>Italic</em>
<i>Italic</i>
<u>Underline</u>
<sub>Subscript</sub>
<sup>Superscript</sup>
<mark>Highlight</mark>
<img src="image.jpg" alt="Image" />
<br />
<code>Code</code>
<kbd>Keyboard</kbd>
<samp>Sample</samp>
<var>Variable</var>
<time>Time</time>
<input type="text" />
<label>Label</label>
<button>Button</button>

Live Output:

Span text, Link, Strong, Bold, Italic, Italic, Underline, Subscript, Superscript, Highlight

Image
Line break

Code, Keyboard, Sample, Variable, Time

Explanation: Inline elements are used for text formatting, small components, or inputs. Self-closing inline tags include <img>, <br>, <input>.
