Ultimate HTML Text Formatting Tags Guide

1. <b> vs <strong>

<b>This is bold</b>
This is bold
<strong>This is strong</strong>
This is strong
<b> — makes text bold only for visual appearance. It is not semantic, meaning it does not tell the browser or screen reader that the text is important — just boldens it.

<strong> — also makes text bold, but it has semantic meaning. It tells that the content is important or urgent. ✅ Use <strong> when the text needs importance; <b> for simple bold styling.

2. <i> vs <em>

<i>Italic Text</i>
Italic Text
<em>Emphasized Text</em>
Emphasized Text
<i> — makes text italic for visual style (e.g., names, thoughts, foreign words). <em> — means “emphasized” — conveys stress or importance. Screen readers even change tone for <em> tags. ✅ Use <i> for appearance, <em> for semantic emphasis.

3. <u> Underline <mark> Highlight

<u>Underlined text</u>
Underlined text
<mark>Highlighted text</mark>
Highlighted text
<u> — simply underlines text; it’s non-semantic (used rarely, may confuse with links). <mark> — highlights text to show importance or relevance (e.g., search results). ✅ Prefer <mark> for emphasis highlighting.

4. <small> Small Text

<small>This is small text</small>
This is small text
Displays text slightly smaller, often used for fine print, disclaimers, or copyright notices. Example: © 2025 All Rights Reserved.

5. <del> Deleted <ins> Inserted

<del>Old text</del> <ins>New text</ins>
Old text New text
<del> shows removed text with a strike line. <ins> shows newly added text. Commonly used in document revisions or change tracking.

6. <sub> Subscript <sup> Superscript

H<sub>2</sub>O
H2O
x<sup>2</sup>
x2
<sub> — lowers text (for chemical formulas, footnotes). <sup> — raises text (for exponents, ordinals). Example: E = mc2

7. <abbr> Abbreviation

<abbr title="HyperText Markup Language">HTML</abbr>
HTML
Defines an abbreviation. The title attribute shows full form on hover. ✅ Improves accessibility and SEO.

8. <q> Short Quote <blockquote> Long Quote

<q>Practice makes perfect</q>
He said, Practice makes perfect.
<blockquote cite="https://example.com">
"This is a long quotation."
</blockquote>
"This is a long quotation."
<q> — short inline quote (browser adds quotes automatically). <blockquote> — block-level quote, used for longer citations. You can add cite="" to reference the source.

9. <cite> Citation

<cite>The Great Gatsby</cite>
The Great Gatsby
Defines the title of a work (book, film, artwork, article). ✅ Example: Avengers: Endgame (movie title).

10. <code> <pre>

<code>console.log("Hello");</code>
Use console.log("Hello");
<pre>
function greet() {
  console.log("Hello World");
}
</pre> 
function greet() {
  console.log("Hello World");
}
<code> — inline code (small code fragments). <pre> — preserves spaces and line breaks (for larger blocks). ✅ Usually used together for displaying code examples.

11. <kbd> Keyboard Input <samp> Output

<kbd>Ctrl</kbd>+<kbd>S</kbd>
Press Ctrl + S to save
<samp>Hello User</samp>
Program output: Hello User
<kbd> — represents keys or user input (keyboard). <samp> — shows sample output from a computer or program. ✅ Used in tech documentation or tutorials.

12. <var> Variable <s> Strikethrough

Area = <var>length</var> × <var>width</var>
Area = length × width
<s>Old price</s> New price
Old price New price
<var> — represents mathematical or programming variables. <s> — shows text as no longer accurate or relevant. Example: ₹1000 ₹800 (discounted).
