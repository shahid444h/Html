HTML Lists – Complete Guide

Introduction to HTML Lists

Lists help organize related information in a structured and readable way. There are three main types:

  • Ordered List (<ol>): Numbered or lettered items.
  • Unordered List (<ul>): Bulleted items.
  • Description List (<dl>): Used for terms and definitions.

1. Ordered List (<ol>)

Definition: Ordered lists display items in a specific order, usually numbered or lettered.

Attributes:

  • type → 1 (numbers), A (uppercase letters), a (lowercase), I (uppercase Roman), i (lowercase Roman)
  • start → sets the starting number
  • reversed → reverses the numbering order
<ol type="A" start="3" reversed>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>
  1. First Item
  2. Second Item
  3. Third Item

Explanation: The list starts from "C" because of start="3" and counts backward due to reversed.

2. Unordered List (<ul>)

Definition: Unordered lists are for items without a specific order, displayed with bullet symbols.

Attribute: type – defines bullet style (disc, circle, square). Modern practice uses CSS instead.

<ul type="square">
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>
  • Apples
  • Bananas
  • Cherries

Explanation: The bullet style here is square. You can also use CSS to customize bullet icons.

3. List Item (<li>)

Definition: Each item in an ordered or unordered list is wrapped in an <li> tag.

Attribute: value → sets a custom number in ordered lists.

<ol>
  <li value="10">Custom Item</li>
  <li>Next Item</li>
</ol>
  1. Custom Item
  2. Next Item

Explanation: The first list item starts at number 10 using value="10".

4. Description List (<dl>)

Definition: Description lists are used for terms and their corresponding definitions or details.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

Explanation: <dt> defines the term and <dd> gives its description.

5. Nested Lists

Definition: You can nest lists inside list items to create sublists.

<ol>
  <li>Fruits
    <ul type="circle">
      <li>Apple</li>
      <li>Mango</li>
    </ul>
  </li>
</ol>
  1. Fruits
    • Apple
    • Mango
