HTML Multimedia – Audio & Video

Learn how to embed and control sound and video files directly in web pages.

1. What is Multimedia in HTML?

Multimedia in HTML allows you to add **audio (sound)** and **video (visuals)** directly into your webpage using the <audio> and <video> elements. These tags **do not require external plugins** like Flash — they are native to HTML5.

Syntax:

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
</audio>

<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>

2. HTML Audio

The <audio> tag embeds sound content such as music, narration, or effects. It supports file formats like MP3, WAV, and OGG.

a) Basic Audio Example

<audio controls>
  <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
</audio>

Explanation: The controls attribute adds play, pause, and volume buttons. The source tag defines the file and its format.

b) Important Attributes of <audio>

  • controls – Displays audio control buttons (Play, Pause, Volume).
  • autoplay – Automatically starts playing when the page loads.
  • loop – Repeats the audio indefinitely.
  • muted – Starts audio with sound muted.
  • preload – Controls how much audio file is loaded:
    • auto – Entire audio loads on page load.
    • metadata – Only loads file info (duration, etc.).
    • none – Loads nothing until user clicks play.

c) Audio with Attributes Example

<audio controls autoplay loop muted>
  <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
</audio>

3. HTML Video

The <video> element allows you to display movies or video clips. Supported formats include MP4, WebM, and OGG.

a) Basic Video Example

<video width="320" height="240" controls>
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
</video>

Explanation: The controls attribute shows playback buttons. width and height define video size.

b) Important Attributes of <video>

  • controls – Displays control buttons (Play, Pause, Volume, Fullscreen).
  • autoplay – Video plays automatically once the page loads.
  • loop – Repeats the video continuously.
  • muted – Starts video muted by default.
  • poster – Shows an image before video starts (thumbnail).
  • preload – Controls loading behavior (auto, metadata, none).

c) Video with Attributes Example

<video width="320" height="240" controls autoplay loop muted
 poster="https://www.w3schools.com/html/pic_trulli.jpg">
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
</video>

4. Best Practices

  • Always provide the controls attribute for user control.
  • Use muted autoplay together because most browsers block autoplay with sound.
  • Optimize audio/video file sizes for faster page load.
  • Add subtitles or captions using <track> for accessibility.
  • Provide multiple file formats (.mp4, .webm, .ogg) for compatibility.
