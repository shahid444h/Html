HTML Iframes – Ultimate Complete Guide with Examples

1. Definition

The <iframe> (Inline Frame) tag is used to embed another HTML document or external website within your current page. It behaves like a small “browser window” inside your main page.

Syntax:
<iframe src="URL" width="600" height="400"></iframe>

2. Basic Iframe Example

Embedding an example page:

<iframe src="https://www.w3schools.com" width="600" height="400"></iframe>

Explanation: The src specifies the webpage to embed. width and height set the visible area of the iframe.

3. Important Attributes of <iframe>

  • src – URL of the page to embed (Required)
  • width & height – Define iframe size in pixels or percentage
  • title – Describes the iframe content for screen readers
  • frameborder – Controls border visibility (deprecated, use CSS)
  • allowfullscreen – Enables fullscreen mode
  • loading – Lazy or eager loading of iframe
  • name – Used for targeting links to open inside iframe
  • sandbox – Restricts iframe actions for security

4. Example with All Attributes

<iframe src="https://www.w3schools.com" width="600" height="400" title="W3Schools Example" frameborder="0" allowfullscreen loading="lazy" name="exampleFrame" sandbox="allow-scripts allow-same-origin"></iframe>

Explanation: This iframe has lazy loading, no border, supports fullscreen, allows scripts, and same-origin access — making it secure and optimized.

5. Target Example – Opening Links inside an Iframe

The name attribute is used to load link pages into a specific iframe. The target in the link matches that name.

<a href="https://www.w3schools.com/html/" target="exampleFrame">Open HTML Tutorial</a>
Open HTML Tutorial in Iframe
Clicking the above link loads the page inside the iframe above.

6. Parent and Top Frame Targets

Iframes can interact with parent or top pages using special target values.

  • _parent – Opens the link in the parent frame (one level up).
  • _top – Breaks out of all iframes and opens the link in the full window.

Example for _parent

<a href="https://www.wikipedia.org" target="_parent">Open in Parent Frame</a>

Output: When this code is inside an iframe and clicked, the link opens in its parent page instead of within the same iframe.

Example for _top

<a href="https://www.google.com" target="_top">Open in Full Window</a>

Output: This link opens the destination in the entire browser window, breaking out of any nested iframes.

7. Embedding a YouTube Video

<iframe width="560" height="315" src="https://www.youtube.com/embed/tgbNymZ7vqY" title="YouTube video player" frameborder="0" allowfullscreen></iframe>

8. Responsive Iframe (100% Width)

<iframe src="https://www.example.com" style="width:100%; height:400px; border:none;"></iframe>

9. Security – Using Sandbox Attribute

The sandbox attribute adds restrictions such as blocking scripts, forms, and pop-ups for security. You can allow specific actions using values like allow-scripts, allow-forms, or allow-same-origin.

<iframe src="demo.html" sandbox="allow-scripts allow-forms"></iframe>

10. Best Practices

  • Always use the title attribute for accessibility.
  • Prefer loading="lazy" to improve performance.
  • Restrict with sandbox for safety.
  • Use CSS for responsiveness instead of fixed pixel widths.
  • Avoid overusing iframes – they increase load time.
