HTML Data Attributes – Interactive Guide

1. What are Data Attributes?

Data attributes allow you to store custom data on HTML elements. They start with data- and can have any name.

Explanation: Useful to store extra information directly in HTML for JavaScript to access, without affecting the page layout or semantics.

2. Syntax

<element data-name="value">Content</element>
Explanation: Replace element with any HTML tag. Replace data-name with your custom attribute name.

3. Example: Storing Custom Data

<button id="btn1" data-user="101" data-role="admin">
  Click Me
</button>
Click the button to see JavaScript read its data-user and data-role.

4. Accessing Data Attributes in JS

const button = document.getElementById("btn1");
console.log(button.dataset.user); // 101
console.log(button.dataset.role); // admin

Click the button above to see output here.

5. Multiple Elements Example

<div class="product" data-id="1" data-price="50">Product 1</div>
<div class="product" data-id="2" data-price="80">Product 2</div>
Product 1
Product 2
Hover over products to see their data in the console, and click to see a dynamic alert.

6. Form Using Data Attributes

<input type="text" data-maxlength="10" id="username" placeholder="Username">
<button id="checkBtn">Check Length</button>

This form shows how you can store metadata (like max length) in data attributes and use JS to validate dynamically.

7. Key Points

  • Data attributes always start with data-.
  • Access in JS via element.dataset.attributeName.
  • Perfect for storing extra info for scripts.
  • Do not use for CSS styling; use classes instead.
