HTML Links – Complete & Detailed Notes

Learn everything about creating hyperlinks in HTML5 with examples, syntax, and outputs.

HTML Document Structure (Quick Recap)

Before learning about links, let’s recall the structure of a basic HTML5 document.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
Explanation:
  • <!DOCTYPE html> → Declares the document as HTML5. Without it, browsers may use quirks mode, causing CSS and layout issues.
  • <html lang="en"> → Defines language (used by screen readers & SEO).
  • <meta charset="UTF-8"> → Ensures all characters & symbols display correctly.
  • <meta name="viewport"> → Makes page responsive on all devices.
  • <title> → Sets the page title shown in the browser tab.

1. Basic Link

The <a> (anchor) tag creates hyperlinks to other pages, files, or locations.

<a href="https://www.example.com">Visit Example.com</a>
Visit Example.com

Explanation: The href attribute defines the URL destination. The text between the tags becomes clickable.

2. Open Link in New Tab

<a href="https://www.example.com" target="_blank">Open in new tab</a>
Open in new tab

Explanation: target="_blank" opens the link in a new tab or window.

3. Download File Link

<a href="sample.pdf" download>Download PDF</a>
Download PDF

Explanation: The download attribute forces the browser to download the file instead of opening it.

4. rel Attribute (Relationship)

The rel attribute describes the relationship between the current page and the linked one.

ValueMeaning
nofollowPrevents search engines from following the link.
noopenerStops new tab from accessing the opener page (security).
noreferrerPrevents sending referrer info to the target site.
authorSpecifies that the linked page is the author of this document.
externalMarks the link as an external website.
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Secure External Link</a>
Secure External Link

5. Email Link

<a href="mailto:someone@example.com">Send Email</a>
Send Email

Explanation: Opens the user’s default mail app with the given email address.

6. Phone Link

<a href="tel:+1234567890">Call Us</a>
Call Us

Explanation: On mobile devices, this opens the phone dialer with the number prefilled.

7. Internal / Anchor Link

<a href="#section2">Go to Section 2</a>

<h2 id="section2">Section 2</h2>
<p>This is the target section.</p>
Go to Section 2

Section 2

This is the target section.

8. Attributes Summary for <a> Tag

Attribute Values Description
hrefURL / #id / mailto / telDefines link destination.
target_self, _blank, _parent, _topSpecifies where to open the link.
relnofollow, noopener, noreferrer, etc.Defines relationship between documents.
downloadfilename (optional)Forces file download.
titleTooltip textDisplays tooltip on hover.
id/classcustomUsed for CSS or JS targeting.

9. Best Practices

  • Use descriptive link text (avoid “click here”).
  • Use target="_blank" with rel="noopener noreferrer" for external links.
  • Keep URLs valid and secure (use HTTPS).
  • Use mailto: and tel: wisely — test them on devices.
  • Style links using CSS pseudo-classes: :hover, :active, :visited.
