HTML Images – Ultimate Guide

Learn everything about displaying and styling images using HTML.

1. What are Images in HTML?

The <img> tag is used to embed images in a webpage. It is a **self-closing tag** (doesn’t need a closing tag). Minimum required attributes: src (source) and alt (alternative text). 

<img src="image.jpg" alt="Description of image">

2. Important Attributes of <img>

a) src – Source of Image

The src (source) attribute defines the image path or URL. It tells the browser where to find the image. 

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="Example Image">

b) alt – Alternative Text

The alt attribute specifies text to display if the image fails to load. It is **important for accessibility** (screen readers read this for visually impaired users) and **SEO**. 

<img src="wrongname.jpg" alt="This is an alternate text">

c) width & height – Image Dimensions

Define the image size in pixels. It’s better to use **CSS for responsive images**, but HTML attributes also work for fixed sizes. 

<img src="https://www.w3schools.com/images/w3schools_green.jpg" width="200" height="100" alt="Resized Image">

d) title – Tooltip Text

Displays a tooltip when you hover over the image. It helps describe the image in brief. 

<img src="https://www.w3schools.com/images/w3schools_green.jpg" title="This is a tooltip" alt="Tooltip Example">

e) loading – Lazy Loading

Controls when an image loads. Values:

  • lazy – loads only when image appears in view (saves data)
  • eager – loads immediately
  • auto – browser decides automatically

<img src="https://www.w3schools.com/images/w3schools_green.jpg" loading="lazy" alt="Lazy Loaded Image">

3. Semantic Images – <figure> and <figcaption>

The <figure> element groups media content (like images or videos) with a caption. The <figcaption> tag provides a description or title for that image. These are **semantic elements** that improve accessibility and SEO. 

<figure>
  <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="HTML Image Example">
  <figcaption>Figure 1: W3Schools Logo</figcaption>
</figure>

4. Best Practices

  • Always use descriptive alt text for accessibility.
  • Use relative paths for local images, absolute URLs for external images.
  • Optimize image size to improve loading speed.
  • Use figure and figcaption for meaningful captions.
  • Use loading="lazy" for performance optimization.
