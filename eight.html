HTML Tables – Complete Notes

1. Definition

Tables in HTML are used to represent data in a structured form of rows and columns. The basic table structure is created using:

  • <table> – Defines the table.
  • <tr> – Table Row (each horizontal row).
  • <th> – Table Header cell (bold, centered by default).
  • <td> – Table Data cell (normal data cell).

📘 Basic Syntax

<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Row 1 Data 1</td>
    <td>Row 1 Data 2</td>
  </tr>
</table>

💡 Output:

Heading 1 Heading 2
Row 1 Data 1 Row 1 Data 2

Explanation: <th> cells appear bold and centered automatically, while <td> are normal data cells.

2. Important Table Attributes

  • border – Defines the border thickness (e.g. border="1").
  • cellpadding – Space between cell content and cell border (e.g. cellpadding="10").
  • cellspacing – Space between individual table cells.
  • width and height – Set the table dimensions.
  • bgcolor – Sets the background color of the table.
  • align – Aligns the table (left, center, right).

📘 Example – Table with Attributes

<table border="2" cellpadding="10" cellspacing="5" width="70%" bgcolor="#f9f9f9" align="center">
  <tr>
    <th>Product</th>
    <th>Quantity</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Laptop</td>
    <td>5</td>
    <td>$5000</td>
  </tr>
  <tr>
    <td>Mouse</td>
    <td>10</td>
    <td>$200</td>
  </tr>
</table>

💡 Output:

Product Quantity Price
Laptop 5 $5000
Mouse 10 $200

Explanation: This table is centered with custom padding, spacing, and background color for better readability.

3. Rowspan and Colspan

These attributes are used to merge multiple cells either vertically or horizontally.

  • rowspan – Merges cells vertically (across rows).
  • colspan – Merges cells horizontally (across columns).

📘 Example:

<table border="1">
  <tr>
    <th rowspan="2">Name</th>
    <th colspan="2">Marks</th>
  </tr>
  <tr>
    <th>Math</th>
    <th>Science</th>
  </tr>
  <tr>
    <td>John</td>
    <td>80</td>
    <td>90</td>
  </tr>
</table>

💡 Output:

Name Marks
Math Science
John 80 90

Explanation: “Marks” spans across 2 columns (colspan), and “Name” covers 2 rows (rowspan).

4. Table Sections – thead, tbody, tfoot

Large tables can be divided into three logical parts:

  • <thead> – Header section (column headings).
  • <tbody> – Body section (main data rows).
  • <tfoot> – Footer section (summary or totals).

📘 Example:

<table border="1">
  <thead>
    <tr>
      <th>Month</th>
      <th>Sales</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>$1000</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$1200</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>$2200</td>
    </tr>
  </tfoot>
</table>

💡 Output:

MonthSales
January$1000
February$1200
Total$2200

Explanation: Dividing tables semantically helps accessibility, readability, and styling using CSS.

5. Caption Tag

The <caption> tag adds a title to your table, displayed above it by default.

📘 Example:

<table border="1">
  <caption>Student Scores</caption>
  <tr><th>Name</th><th>Score</th></tr>
  <tr><td>Ava</td><td>95</td></tr>
</table>

💡 Output:

Student Scores
NameScore
Ava95

Explanation: The <caption> provides a short description for the table content.
