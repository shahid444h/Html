HTML IDs & Classes – Complete Guide

1. What are IDs and Classes?

In HTML, IDs and classes are used to identify elements for styling with CSS or manipulation with JavaScript.

  • ID: A unique identifier for a single element. Only one per page.
  • Class: A reusable identifier. Multiple elements can share the same class.
Explanation: Use IDs when you want to target a single element uniquely. Use classes when you want to style or select multiple elements in the same way.

2. ID Example

HTML Code:

<p id="unique-paragraph">This paragraph has an ID "unique-paragraph".</p>

Live Output:

This paragraph has an ID "unique-paragraph".

Explanation: The ID is unique. In CSS: #unique-paragraph { color: red; }. In JavaScript: document.getElementById("unique-paragraph").

3. Class Example

HTML Code:

<p class="highlight">This is a paragraph with class "highlight".</p>
<p class="highlight">Another paragraph sharing the same class.</p>

Live Output:

This is a paragraph with class "highlight".

Another paragraph sharing the same class.

Explanation: Multiple elements can have the same class. CSS: .highlight { background-color: yellow; }. JavaScript: document.getElementsByClassName("highlight").

4. Multiple Classes

HTML Code:

<p class="highlight bold">This paragraph has two classes: "highlight" and "bold".</p>

Live Output:

This paragraph has two classes: "highlight" and "bold".

Explanation: Assign multiple classes by separating them with spaces. Each class can apply its own style.
