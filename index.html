HTML Structure Explained

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

<!DOCTYPE html>: Declares that this document is HTML5. Without it, the browser goes into quirks mode 1990s Html, and CSS might not render properly.

<!DOCTYPE html>

<html lang="en">: The root element. The lang attribute tells browsers and screen readers the page language. Missing it may reduce accessibility and SEO.

<html lang="en"> ... </html>

<head>: Contains metadata like page title, character set, CSS links, and scripts. Not visible on the page but essential for proper page rendering and SEO.

<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>

<meta charset="UTF-8">: Ensures characters (including emojis, accented letters) are displayed correctly. Without it, some text may appear as gibberish.

<meta charset="UTF-8">

<meta name="viewport" ... >: Makes the page responsive on mobiles and tablets. Without it, the page might appear zoomed out or not scale properly.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
